## 拼车/上车/来车

### 前情提要

GitHub Copilot 是一款由 GitHub 和 OpenAI 合作推出的人工智能编程助手工具，于 2022 年 8 月 22 日开始收费。社区热佬随后开发了一个小工具，其中包含了'上车'、'拼车'等功能，解决了用户需要到处发送 ID 来拼车的问题。

### 时至今日

如今，`拼车`、`上车`、`来车`已经成为社区内的一个流行用语，通常用来表示一起分享免费的服务和资源。
