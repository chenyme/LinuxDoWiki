## 使用说明

<Callout type='info'>
    本服务的使用方式与 OpenAI 官方 API 完全一致。
</Callout>

1. 访问 <Link href='https://api.oaipro.com/' target='_blank' style={{ textDecoration: 'none', color: '#3A9FE8' }}>api.oaipro.com</Link>，使用 Linux Do Connect 登录。

2. 由于本服务为 Neo 提供的公益项目，暂不提供试用。如需使用请前往 '钱包' 页面进行充值。

3. 充值完成后，进入 **'令牌'** 页面，点击 **'添加令牌'**，即可获取您的专属 API Key。

- 您可以前往 **'聊天'** 页面 - 点击左上角头像 - **'应用设置'** - **'语言模型'**，填入刚获取的 API Key，并将 API 代理地址设置为 `https://api.oaipro.com/v1`，即可开始使用。

- 同时支持通过 Curl、Python、Java 等方式调用 API。调用方法与 OpenAI 官方一致，只需将 base_url 参数设置为 `https://api.oaipro.com/v1`，api_key 替换为您的专属 API Key 即可。
