# HTTP 测试接口

<br></br>

<Link href="https://http.rw/" target="_blank" className="text-blue-500">http.rw</Link>

这是一个使用 httpbin 的开源程序实现的 HTTP 测试接口。由于目前这个程序比较老，加上套了一层 CF 效果可能会不太理想，回头会逐步替换为 PHP 的。

你打开此网页就是个接口列表和测试工具，可以玩玩。当然，在命令行使用也毫无问题，如：

```shell copy "
curl http.rw/ip
```

这样就可以获取本机IP，大概就是这样。
