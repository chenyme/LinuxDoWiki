# 通配子域 Fuclaude

与 OAIFree 的通配子域名功能类似，`*.fuclaude.oaifree.com` 现已支持通配子域名访问登录 Fuclaude 镜像（注：**非 `fuclaude.oaifree.com`**）。您可以使用 <Link href='https://linuxdo.fuclaude.oaifree.com' target='_blank' style={{ textDecoration: 'none', color: '#3A9FE8' }}>linuxdo.fuclaude.oaifree.com</Link>、<Link href='https://oaifree.fuclaude.oaifree.com' target='_blank' style={{ textDecoration: 'none', color: '#3A9FE8' }}>oaifree.fuclaude.oaifree.com</Link> 等子域名来访问镜像服务。

实际上，`*.fuclaude.oaifree.com` 中的 `*` 可以是任何符合域名规则的字符。由于不同域名的 Cookie 相互独立，这在一定程度上可以防止服务被阻断。
