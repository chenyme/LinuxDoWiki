# 社区守则

总的来说，本着'我为人人、人人为我'的原则，真诚、友善地对待他人，不以一己私利为重，就不会有问题。

希望你在这里玩得愉快！

## 社区规则

### 禁止内容

- 政治相关。我们不讨论、评价国家政策及国家领导人。若你在其他平台有此类负面发言，这里也不欢迎你。
- 血腥、暴力、赌博、毒品、涉黑相关。若你在其他平台有此类负面发言，这里也不欢迎你。
- 色情相关。包括擦边内容。
- 恶意引战内容。包括易对立话题。
- 诋毁、侮辱他人的内容。包括挂人带节奏。
- 谣言、欺诈、黑产类内容。
- 未明确标注的病毒、恶意软件及网页等。
- 恶意刷帖、垃圾信息攻击。包括自动发帖、帖子内容无文字意义、滥用彩色文字、隐藏字符等。
- 使用 AI 生成的文字内容。如需发布，请以截图形式。
- 违规的推广、引流内容。具体见下方细则。
- 主动私聊发送骚扰、广告相关内容。
- 社区聊天频道发送打卡类内容。
- 发布以上内容，发布者自行承担法律责任、道德风险，且最高可永久封禁社区账号。

### 避免内容

- 裂变、拉人头类传播内容。具体见下方细则。
- 非必要使用外部实时通讯进行沟通。
- 支付宝红包码、拼多多砍价、虚拟币空投等。
- 自行顶帖防沉。
- 阴阳怪气言语。
- 过于夸大其词，过分标题。
- 原样搬运（非二创），且不标注出处。
- 与讨论主题毫不相关的内容。
- 在求助帖（如：快问快答），让帖主去搜索、问 AI 等无帮助的内容（可帮顶，拉懂佬）。
- 仅有链接或简短内容，就引流到链接、群组。
- 节点、订阅等内容发在非权限区。
- 小尾巴签名使用 gif 动图，拖慢帖子加载速度。
- 发送以上内容，可能会临时封禁。

### 账号相关

- 勿外借、买卖账号，避免因此导致账号封禁。
- 尽量设置复杂密码，开启 2FA。避免通用、公用密码，以防被扫、盗号。
- 目前已没有账号清理规则。
- 账号注册需要认真填写注册资料，社区管理将会审核注册资料。
- 新账号注册完很短时间内发布较短内容，可能会风控，管理会解。
- 注册账号、创建、接受邀请需要使用：内地、港、澳、台网络。
- 账号邮箱后缀支持列表：
    
    `126.com`， `139.com`， `163.com`， `189.cn`， `aliyun.com`， `apache.org`， `deepseek.com`， `edu.cn`， `edu.hk`， `edu.mo`， `edu.tw`， `foxmail.com`， `gmail.com`， `gov.cn`， `qq.com`， `sina.cn`， `sina.com`， `sohu.com`， `xiaomi.com`， `yahoo.com`

### AFF 规则

- AFF 内容作为主帖出现，视为推广内容。具体见下方细则。
- AFF 内容作为跟帖时，不得无故出现。
- 跟帖时，需要在链接同行使用 AFF 标签（输入：`#AFF`）。

### 公益推广

- 适用范围：纯开源且无间接引流项目、纯公益且无间接引流项目。
- 推广方运营内容不得出现：政治、色情、暴力、血腥、赌博、毒品、涉黑、欺诈、侮辱他人等恶意内容。
- 推广内容必须携带 `公益推广` 标签。
- 推广内容原则上每个账号每周不超过 1 篇。
- 如果推广方网站或程序需要登录，必须接入 <Link href='https://connect.linux.do/' target='_blank' className='text-blue-500'>LINUX DO Connect</Link> 且可直接登录。
- 个人自用、闲置交换、拼车不属于推广范畴。
- 有明显以营利为目的的倒买倒卖行为，属于推广范畴。

### 普通推广

- 适用于非公益推广内容。
- 推广方运营内容不得出现：政治、色情、暴力、血腥、赌博、毒品、涉黑、欺诈、侮辱他人等恶意内容。
- 推广内容必须发布在 `深海幽域` 或其子版块。
- 推广内容必须携带 `推广` 标签。
- 推广内容原则上每个账号每周不超过 1 篇（过多发布容易被用户屏蔽）。
- 如果推广方网站或程序需要登录，必须接入 <Link href='https://connect.linux.do/' target='_blank' className='text-blue-500'>LINUX DO Connect</Link> 且可直接登录。
- 禁止在本社区引流至其他社区，此行为视为不友好行为。

### 高级推广

- 适用于非公益推广内容。
- 推广方运营内容不得出现：政治、色情、暴力、血腥、赌博、毒品、涉黑、欺诈、侮辱他人等恶意内容。
- 必须拥有并佩戴 <Link href='https://linux.do/s/subscriptions/' target='_blank' className='text-blue-500'>'富可敌国'</Link> 头衔（通过订阅获得）。
- 推广内容可发布在 `深海幽域` 以外版块。
- 推广内容必须携带 `高级推广` 标签。
- 推广内容原则上每个账号每日不超过 1 篇（过多发布容易被用户屏蔽）。
- 头衔失效后，该账号所有推广帖将被删除。
- 推广内容作为回复时，不得无故出现。
- 禁止在本社区引流至其他社区，此行为视为不友好行为。

### 定制推广

- 社区近30天（250406-250505）独立访客数：640K，页面浏览量：42M。
- 数据来源于 Google Analytics。
- 请根据实际情况合理评估。

- 社区收益暂不公开，以防不法分子觊觎。
- 为社区提供充足的 `claude`、`gpt`、`gemini` 等高级 AI 模型服务。
- 购置家用宽带代理为社区现有及未来项目提供支持。
- 为社区成员提供 `@linux.do` 邮箱服务。
- 支持社区服务器等基础设施运营支出。
- 对社区成员优质项目进行激励和孵化。

### 内容创作

- 发布后 5 分钟内且无人回复的帖子可自行删除。
- 超出时限仅可通过举报并说明理由联系管理员删除。
- 社区有权直接使用、编辑或删除您的帖子，详见 <Link href='https://linux.do/tos/' target='_blank' className='text-blue-500'>服务条款</Link>。
 
### 资源网盘

- 资源网盘帖子必须包含对应网盘的标签。
- 多个网盘需选择多个对应标签，不可遗漏。
- 标签请参考 <Link href='https://linux.do/tos/' target='_blank' className='text-blue-500'>网盘分组</Link>，如有缺失请联系管理员添加。
- 资源网盘必须支持主流浏览器直接访问，禁止强制仅限 APP 使用。
- 一般网盘资源分享不属于推广范畴。

### 版块说明

- 每个版块均有对应的：无权限、1级权限、2级权限、3级权限子版。
- 权限子版不根据专业程度设定，通常根据保密、敏感程度来定。
- 社区鼓励佬友们发帖时考虑、正确选择版块：
    - **开发调优**：开发、测试、调试、部署、优化等相关内容。
    - **资源荟萃**：软件分享、开源仓库、视频课程、书籍等资源分享。
    - **文档共建**：默认为wiki格式，共同编写各类文档。
    - **跳蚤市场**：交易相关版块，包含实体和虚拟物品供求、拼车等。
    - **非我莫属**：求职版块，包括招聘和求职等供需信息。
    - **读书成诗**：分享读书笔记和心得体会。
    - **扬帆起航**：创业相关版块，展示创业历程和成果。
    - **前沿快讯**：科技资讯、行业动态等新闻类内容。
    - **福利羊毛**：分享优惠活动、抽奖等福利信息。
    - **搞七捻三**：日常闲聊话题，包括衣食住行、影音游戏等。
    - **深海幽域**：轻松娱乐版块，包含趣味话题，不上信息流。
    - **运营反馈**：社区反馈建议和公告通知。
- 由于分类较为宽泛，请使用 <Link href='https://linux.do/tos/' target='_blank' className='text-blue-500'>标签</Link> 对帖子进行细分归类。
- 版块等同于父类继承，标签则是接口实现。
- 可以选择相应版块、标签进行筛选、关注、屏蔽。
- 首页信息流按回帖时间倒序排列，具体版块内按创建时间倒序排列。

### 关于举报

- 举报是匿名的，社区永远不会透露举报人信息。
- 举报是共建、共同维护社区的有效方式。
- 被举报的帖子需要管理审批处理。
- 具体的审批方式说明：
    - 同意类（包括：同意但不处理，同意且隐藏，同意且删除等），被举报人会收到处置通知。
    - 不同意类，被举报人不会收到通知。
    - 忽略类（包括：忽略且不操作，忽略且删除），被举报人会收到处置通知。
- 只有同意类的举报处理才会影响被举报人的统计数据。
- 被举报人对举报处理存在异议时，请优先 <Link href='https://linux.do/about/' target='_blank' className='text-blue-500'>联系管理</Link> 反馈。

### 我们收集哪些信息？

- 当您在我们的网站上注册时，我们会收集您的信息，当您通过阅读、撰写和评价本论坛上分享的内容参与本论坛时，我们会收集您的数据

- 在注册和发帖时，我们会记录帖子所来自的 IP 地址。我们还可能保留服务器日志，其中包含向我们的服务器发送的每个请求的 IP 地址

- 发送电子邮件通知 — 您提供的电子邮件地址可能被用于向您发送信息、您请求的关于话题变更的通知、回应您的用户名、回复查询或其他请求
