## 功能介绍

Chat2API - OAIFree 是始皇对 OpenAI 相关接口进行转换，模拟 API 调用格式，把 access token 传进去就开转，可以实现比如对话、tts、whisper、画图等 API 调用方式。

- 只为ChatGPT Plus提供服务。
- 目前频控 10/10s，将来一定会跟 LINUX DO Connect 挂钩。
- 限制了 whisper 上传文件大小为：4M。
- 支持使用 gizmo，使用 g-xxx 的 id 作为模型名即可。
- Team 账号只需要传递 ChatGPT-Account-ID 头即可实现切换。

而这一切，只需要使用 **ChatGPT Plus** 账号的 Access Token 即可调用！

## 如何调用

调用方式和 OpenAI API 调用一模一样，只需要将 Base_url 换为 OAIFree 的 API 地址即可, Api_key 换为 Access Token。
- Team 账号只需把 `<access token>` 替换成 `<access token>,<team account id>`
